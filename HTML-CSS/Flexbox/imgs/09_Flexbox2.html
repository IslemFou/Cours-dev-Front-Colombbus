Cours FlexBox

A - Comportement normal

Le principe de la mise en page avec FlexBox est simple : vous définissez un contenaire et à l'intérieur de ce contenaire vous placez plusieurs éléments.
La création d'un contenaire permet de s'ajuster et appliquer les règles de FlexBox par rapport aux éléments à l'intérieur.

C - Direction de Flex

FlexBox nous permet d'organiser ses éléments dans le sens désiré. Avec flex-direction on peut positionner nos éléments d'une façon par exemple : verticale : column.
Flex-direction peut prendre les valeurs suivantes:
row : les éléments s'organisent en horizontale (par défaut).
column : les éléments s'organisent en verticale.
row-reverse : les élément s'alignent vers le sens horizontal mais inversé l'ordre des éléments.
Column-reverse : les élément s'alignent vers le sens vertical mais l'ordre est inversé.

D - Les Wrap : retour à la ligne

La propriété flex-wrap indique si les éléments flexibles sont contraints à être disposés sur une seule ligne ou s'ils peuvent être affichés sur plusieurs lignes avec un retour automatique. Si le retour à la ligne est autorisé, la propriété permet également de contrôler la direction dans laquelle les lignes sont empilées.
Les valeurs:
Valeurs avec un mot-clé
flex-wrap: nowrap; Valeur par défaut
flex-wrap: wrap;
flex-wrap:wrap-reverse;
Valeurs globales
flex-wrap: inherit;
flex-wrap: initial;
flex-wrap: unset;
